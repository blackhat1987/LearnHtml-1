CSS 分组 和 嵌套 选择器

分组选择器

在样式表中有很多具有相同样式的元素。
h1
{
    color:green;
}
h2
{
    color:green;
}
p
{
    color:green;
}
为了尽量减少代码，你可以使用分组选择器。
每个选择器用逗号分隔。
在下面的例子中，我们对以上代码使用分组选择器：
实例
h1,h2,p
{
    color:green;
}
